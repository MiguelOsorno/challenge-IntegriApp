# Angular Web

Aplicación hecha en Angular



## Instalación

Ingresa a la carpeta Backend y ejecuta el siguiente comando

```bash
npm install -g json-server
```

```bash
npm install  
```

Ingresa a la carpeta AngularWeb y ejecuta el siguiente comando

```bash
npm install
```

## Ejecucion

Ingresa a la carpeta Backend y ejecuta el siguiente comando

```bash
npm start
```
Ingresa a la carpeta AngularWeb y ejecuta el siguiente comando

```bash
ng serve -o
```
